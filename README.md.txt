# Daily Reflection Tree

This is a deterministic reflection system based on three axes:

1. Locus of Control
2. Contribution vs Entitlement
3. Radius of Concern

## Features
- No AI at runtime
- Fully deterministic
- Fixed options only
- Signal-based branching

## Files
- reflection-tree.json → main tree
- tree-diagram.md → flow diagram
- transcripts → sample paths
- write-up.md → design explanation
